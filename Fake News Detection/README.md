Fake News Detection System
--------------------------

Description: 

The Fake News Detection System is a machine learning project designed to classify news articles as either fake or genuine. It utilizes Natural Language Processing (NLP) techniques to preprocess text data and various classification algorithms to predict the veracity of news content. 

Features: 
- Data Preprocessing: Cleans and normalizes the text data by removing URLs, HTML tags, punctuation, digits, and newline characters. 

- Model Training: Implements multiple machine learning models, including Logistic Regression, Decision Tree Classifier, Random Forest Classifier, and Gradient Boosting Classifier.

- Prediction Capabilities: Allows users to input a news article for manual testing and get predictions from different classifiers.

- Performance Metrics: Provides accuracy, precision, recall, and F1-score metrics to evaluate model performance. 

Technologies Used: 
- Python 
- Pandas
- NumPy
- Scikit-learn
- TfidfVectorizer
- Jupyter Notebook
