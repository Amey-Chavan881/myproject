# Movie Recommendation Model

## Description

The Movie Recommendation Model is a machine learning project aimed at suggesting movies based on user preferences. It leverages Natural Language Processing (NLP) techniques to analyze movie descriptions and genres, providing personalized recommendations by comparing similarities between movies.

## Features

- **Data Preprocessing:** Reads and cleans the movie dataset, combining genre and overview into a single tags feature for enhanced similarity calculations.

- **Recommendation Engine:** Utilizes cosine similarity to recommend movies based on the user-selected title, finding the top similar movies from the dataset.

- **Interactive Testing:** Users can input a movie title to receive a list of recommended movies based on their chosen title.

- **Performance Metrics:** Effectively handles large datasets (10,000 entries) and returns accurate recommendations based on similarity metrics.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
